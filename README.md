# Prompt para generar la interfaz Bandera de México

## Instrucciones

Genera una interfaz web que represente la Bandera de México utilizando tres archivos separados llamados:

- index.html
- style.css
- script.js

## Requisitos

1. La página debe mostrar un título centrado que diga: "Bandera de México".
2. Debe haber tres secciones superiores:
   - Esperanza (verde) con un input tipo range para manipular el color verde.
   - Unidad (blanco) en el centro (sin slider).
   - Sangre de nuestros héroes nacionales (rojo) con un input tipo range para manipular el color rojo.
3. Debajo de cada slider debe mostrarse el valor hexadecimal.
4. En la parte inferior deben mostrarse tres rectángulos grandes alineados horizontalmente.
5. En la zona central blanca donde aparecen los caracteres "@@@@@", se debe colocar el escudo de la bandera de México centrado.
6. Usar flexbox para la distribución.
7. El diseño debe verse moderno y limpio.
8. Mostrar los archivos separados claramente como:

--- index.html ---
--- style.css ---
--- script.js ---
